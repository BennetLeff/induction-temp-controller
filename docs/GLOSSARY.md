# Glossary

- **SSR** — Solid State Relay, a light-controlled AC switch
- **PID** — Control loop (Proportional, Integral, Derivative)
- **Duty Cycle** — % of time the SSR is ON in a window
- **Optocoupler** — Isolates DC and AC sides
- **Creepage/Clearance** — Minimum safe PCB spacing
- **KiCad** — Open-source PCB design suite
- **MAX31856** — Precision thermocouple amplifier
- **K-Type Probe** — High-temperature sensor
- **Induction Cooker** — Appliance using magnetic fields to heat metal pans
