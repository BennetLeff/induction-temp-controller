# Safety Notes

### ⚠️ High Voltage Warning
This project involves switching **120–240V AC**.
You MUST:

- Know basic AC safety
- Keep AC and DC isolated
- Use properly rated SSR
- Fuse the AC input
- Always ground the enclosure
- Never touch AC while powered
- Use an enclosure with strain relief

---

## Layered Safety Model

1. **SSR optical isolation**
2. **Cooker internal safety circuits**
3. **Fuse**
4. **Grounded chassis**
5. **GFCI outlet recommended**

---

### If you're unsure about AC, STOP and ask.
